# Redis-Case-and-Queue-Example
Redis增删改查以及队列示例
PHP客户端使用的是 [Predis](https://github.com/nrk/predis)

#测试使用
```
git clone git@github.com:jakehu/Redis-Case-and-Queue-Example.git
cd Redis-Case-and-Queue-Example
composer install
```
